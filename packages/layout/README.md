# two-ui layout component

## 安装

```shell
npm install @two-ui/layout
```

## 使用

```javascript
import twoLayout from '@two-ui/layout';

export default {
  components: {
    [twoLayout.name]: twoLayout,
  },
};

```

## 介绍
